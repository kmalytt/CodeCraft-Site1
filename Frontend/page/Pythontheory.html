ТЕМА 1: Основи Python

1.1 Вступ до Python

Python — це високорівнева, проста у вивченні мова програмування. Її використовують у:

  • веб-розробці
  • штучному інтелекті
  • аналізі даних
  • автоматизації
  • створенні ігор
  • робототехніці

Python популярний через простий синтаксис, схожий на людську мову.

Приклад простого коду:

print("Привіт, Python!")

1.2 Встановлення та запуск Python

Щоб почати писати код на Python:

  • Завантажити Python з офіційного сайту: python.org
  • Встановити IDE або редактор коду: VS Code, PyCharm, Jupyter Notebook або IDLE
  • Створити файл з розширенням .py (наприклад hello.py)
  • Запустити код через термінал або IDE
python hello.py

Приклад файлу:

# hello.py print("Привіт, Python!")

1.3 Синтаксис Python

Python має унікальний синтаксис:

  • Не використовує фігурні дужки {} для блоків коду
  • Блоки визначаються відступами (4 пробіли рекомендується)
  • Відступи обов'язкові, неправильний відступ викликає помилку
age = 18 if age >= 18: print("Ви повнолітній")

Неправильно:

if age >= 18: print("Ви повнолітній") # ❌ Помилка IndentationError

1.4 Змінні

Змінні зберігають дані і не потребують оголошення типу (динамічна типізація).

Правила назви змінних:

  • Починаються з літери або _
  • Можуть містити літери, цифри та _
  • Не можна використовувати ключові слова Python (if, for, while тощо)
name = "Alice" age = 25 is_student = True

1.5 Типи даних

Python підтримує різноманітні базові типи даних:

  • int — Цілі числа, приклад: x = 10
  • float — Дробові числа, приклад: y = 3.14
  • str — Рядки, приклад: name = "Python"
  • bool — Булеві значення, приклад: flag = True
  • list — Списки, приклад: fruits = ["apple", "banana"]
  • tuple — Кортежі, приклад: point = (10, 20)
  • set — Множини, приклад: nums = {1,2,3}
  • dict — Словники, приклад: person = {"name":"Alice", "age":25}

1.6 Числа та арифметичні операції

Python підтримує стандартні математичні операції:

  • + — додавання, приклад: 5 + 3 = 8
  • - — віднімання, приклад: 5 - 3 = 2
  • * — множення, приклад: 5 * 3 = 15
  • / — ділення, приклад: 5 / 2 = 2.5
  • // — ціле ділення, приклад: 5 // 2 = 2
  • % — остача від ділення, приклад: 5 % 2 = 1
  • ** — піднесення до степеня, приклад: 2 ** 3 = 8

1.7 Рядки

Рядки — текст, укладений у лапки. Можна об’єднувати, вирізати, форматувати.

text = "Python" print(text[0]) # P print(text.upper()) # PYTHON

1.8 Булеві значення

Булевий тип представляє True або False. Часто використовується в умовах.

is_adult = 17 >= 18 print(is_adult) # False

1.9 Вивід (print) та введення (input)

name = input("Введи ім'я: ") print("Привіт,", name)

1.10 Коментарі

Коментарі — це пояснення коду.

  • Однорядкові — #
  • Багаторядкові — """ ... """
# Це однорядковий коментар """ Це багаторядковий коментар """

ТЕМА 2: Операції та структури даних у Python

Python має потужні інструменти для роботи з даними — від простих операторів до складних структур (списки, словники). У цій темі ти навчишся управляти інформацією так, як це роблять професійні програмісти.

2.1 Арифметичні оператори

  • + додавання
  • - віднімання
  • * множення
  • / ділення (float)
  • // цілочисельне ділення
  • % остача від ділення
  • ** піднесення до степеня
a = 10 b = 3 print(a + b) # 13 print(a / b) # 3.3333 print(a // b) # 3 print(a ** b) # 1000

2.2 Логічні оператори

  • and — І
  • or — Або
  • not — Заперечення
x = 5 print(x > 3 and x < 10) # True print(x < 3 or x == 5) # True print(not (x == 5)) # False

2.3 Оператори порівняння

  • == дорівнює
  • != не дорівнює
  • > більше
  • < менше
  • >= більше або дорівнює
  • < = менше або дорівнює
print(5 == 5) # True print(5 != 3) # True print(5 >= 10) # False

2.4 Списки (list)

Список — змінна колекція даних. Можна додавати, видаляти, змінювати та сортувати елементи.

numbers = [1, 2, 3, 4] numbers.append(5) # додати numbers.remove(2) # видалити 2 numbers[0] = 10 # змінити елемент print(numbers)

2.5 Кортежі (tuple)

Кортеж — незмінний список. Добре підходить для даних, які не повинні змінюватись.

data = (10, 20, 30) print(data[1]) # 20

2.6 Набори (set)

Набір — колекція унікальних значень (без повторів).

nums = {1, 2, 3, 3, 2} print(nums) # {1, 2, 3} a = {1, 2, 3} b = {3, 4} print(a | b) # об’єднання → {1, 2, 3, 4} print(a & b) # перетин → {3}

2.7 Словники (dict)

Словник — це пари ключ:значення. Добре підходить для зберігання структурованих даних.

person = { "name": "Anna", "age": 17 } print(person["name"]) # Anna person["age"] = 18

2.8 Ітератори

my_list = [1, 2, 3] it = iter(my_list) print(next(it)) # 1 print(next(it)) # 2

2.9 Вбудовані функції для структур даних

  • len() — довжина
  • max() — найбільше
  • min() — найменше
  • sum() — сума
  • sorted() — сортування
nums = [5, 1, 10] print(len(nums)) # 3 print(sorted(nums)) # [1, 5, 10]

2.10 Спискові включення (list comprehension)

numbers = [x * 2 for x in range(5)] print(numbers) # [0, 2, 4, 6, 8]

ТЕМА 3: Умови та розгалуження в Python

Умови дозволяють програмі «приймати рішення» — виконувати певний код тільки тоді, коли певна умова істинна.

3.1 Оператор if — основа умов

Оператор if виконує блок коду, якщо умова має значення True.

  • Умова завжди повертає True або False.
  • Відступи обов’язкові — блок коду під if має бути відступлений на 4 пробіли.
  • Якщо умова False, код під if пропускається.
age = 18 if age >= 18: print("Ви повнолітні")

3.2 Оператор else — дія, коли умова False

else виконується тоді, коли всі попередні умови були False.

x = 5 if x > 10: print("Більше 10") else: print("10 або менше")

3.3 Оператор elif — перевірка кількох умов

Використовується, коли є кілька взаємовиключних варіантів. Виконується тільки перший elif, що є True.

score = 75 if score >= 90: print("A") elif score >= 80: print("B") elif score >= 70: print("C") else: print("D")

3.4 Умови з логічними операторами (and, or, not)

Комбінуйте кілька умов для складнішої логіки.

age = 20 has_id = True if age >= 18 and has_id: print("Можна увійти")

3.5 Вкладені умови (if всередині if)

Можна будувати структури з кількома рівнями перевірок.

age = 20 student = True if age >= 18: if student: print("Квиток зі знижкою") else: print("Звичайний квиток")

3.6 Порівняння рядків в умовах

Рядки порівнюються за значенням. Регістр має значення: "Hello" ≠ "hello".

name = "Admin" if name.lower() == "admin": print("Вхід дозволено")

3.7 Умови з числами (>, <, ==, !=, <=, >=)

Числові порівняння — найпоширеніший варіант умов.

x = 15 if 10 <= x <= 20: print("Число в діапазоні 10–20")

3.8 Тернарний оператор — коротка форма if/else

age = 17 status = "Дорослий" if age >= 18 else "Неповнолітній" print(status)

3.9 match-case (аналог switch у Python)

Доступно з Python 3.10. Заміна багатьох elif для вибору з фіксованих значень.

day = 3 match day: case 1: print("Понеділок") case 2: print("Вівторок") case 3: print("Середа") case _: print("Невідомий день")

3.10 Умови в одній лінії

x = 5 if x > 0: print("Позитивне число")

ТЕМА 4: Цикли в Python

Цикли дозволяють виконувати один і той самий код багато разів — це основа автоматизації, обробки даних, роботи зі списками та файлами.

4.1 Цикл for — перебір елементів колекцій

Цикл for проходить по кожному елементу послідовності: список, рядок, кортеж, словник, множина, range().

fruits = ["apple", "banana", "cherry"] for f in fruits: print(f)

4.2 Цикл for + range() — повтори певну кількість разів

range() створює послідовність чисел:

  • range(stop) → 0 … stop-1
  • range(start, stop)
  • range(start, stop, step)
for i in range(5): print(i)

4.3 Перебір рядків циклом for

У Python рядок є послідовністю символів. Це означає, що його можна перебирати циклом for, отримуючи кожен символ окремо. Такий перебір зручно використовувати для аналізу тексту, підрахунку символів, фільтрації, пошуку тощо.

text = "Hello" for char in text: print(char)

4.4 Перебір словника

Словник (dict) зберігає дані у форматі ключ: значення. Через цикл for ми можемо отримувати:

  • лише ключі
  • лише значення
  • пари ключ–значення (через .items())
mydict = {"name": "Alice", "age": 25} # Ключі for key in mydict: print(key) # Значення for value in mydict.values(): print(value) # Ключ + значення for key, value in mydict.items(): print(key, value)

4.5 Цикл while — виконується, поки умова True

Цикл while повторює блок коду доти, доки вказана умова залишається істинною. Важливо змінювати змінні всередині циклу, щоб уникнути нескінченного виконання.

x = 1 while x <= 5: print(x) x += 1

4.6 Нескінчений цикл (while True)

Нескінченний цикл — це цикл, який ніколи самостійно не завершується. Найчастіше використовується у програмах, які повинні постійно чекати на введення користувача або події. Вихід із такого циклу зазвичай роблять через break.

while True: cmd = input("Введіть команду: ") if cmd == "exit": break

4.7 break — зупиняє цикл повністю

Оператор break миттєво завершує виконання циклу (for або while), незалежно від того, чи виконалися всі ітерації. Зручно використовувати при пошуку або для дострокового завершення.

for i in range(10): if i == 5: break print(i)

4.8 continue — пропускає ітерацію

Оператор continue пропускає поточну ітерацію циклу і переходить до наступної. Зручно, коли потрібно пропустити певні значення без зупинки циклу.

for i in range(5): if i == 2: continue print(i)

4.9 else після циклу

У Python цикл for або while може мати блок else. Він виконається ТІЛЬКИ якщо цикл завершився природно (не через break). Це зручно при пошуку значень — можна повідомити, що елемент не знайдено.

for i in range(5): print(i) else: print("Цикл завершено")

4.10 Вкладені цикли

Вкладений цикл — це цикл всередині іншого циклу. Використовується для роботи з матрицями, таблицями, багатовимірними структурами даних або для подвійних переборів (наприклад, комбінацій).

for i in range(3): for j in range(3): print(i, j)

ТЕМА 5: Функції в Python

Функції дозволяють структурувати код, робити його повторно використовуваним та зрозумілішим.

5.1 Створення функції

У Python функції створюють за допомогою ключового слова def. Функція виконує блок коду, який запускається лише після виклику. Назви функцій пишуть у стилі snake_case. Код усередині функції обов’язково має відступ 4 пробіли.

def greet(): """Проста функція, яка нічого не приймає і лише друкує текст.""" print("Привіт, Python!") greet() # виклик функції

5.2 Функція з параметрами

Параметри дозволяють передавати дані всередину функції. Функція може використовувати їх у своїх обчисленнях.

def greet(name): """Функція приймає одне значення — ім’я.""" print(f"Привіт, {name}!") greet("Alice") greet("Bob")

5.3 Функція з поверненням значення

Функція може повернути результат роботи за допомогою return. Після return виконання функції зупиняється.

def add(a, b): """Повертає суму двох чисел.""" return a + b result = add(5, 3) print(result) # 8

5.4 Значення за замовчуванням (default)

Значення за замовчуванням використовується, якщо аргумент не передали. Це зручно для необов’язкових параметрів.

def greet(name="Гість"): """Якщо ім'я не передано — використовується 'Гість'.""" print(f"Привіт, {name}!") greet() # Привіт, Гість! greet("Alice") # Привіт, Alice!

5.5 Функції з необмеженою кількістю аргументів (*args)

*args дозволяє передати у функцію будь-яку кількість позиційних аргументів. Всередині args — це кортеж.

def sum_all(*args): """Підсумовує будь-яку кількість чисел.""" total = 0 for num in args: total += num return total print(sum_all(1, 2, 3, 4)) # 10

5.6 Функції з ключовими аргументами (**kwargs)

**kwargs дозволяє передавати необмежену кількість параметрів у вигляді "ключ = значення". Усередині — це словник.

def greet_all(**kwargs): """Перебирає всі ключові аргументи.""" for name, msg in kwargs.items(): print(f"{name}: {msg}") greet_all(Alice="Привіт!", Bob="Здрастуйте!")

5.7 Анонімні функції (lambda)

lambda — маленька функція без назви. Зазвичай використовується для коротких одноразових операцій.

square = lambda x: x**2 print(square(5)) # 25

5.8 Вбудовані функції Python

Python має багато корисних вбудованих функцій: len(), sum(), max(), min(), sorted() та інші.

nums = [1, 2, 3, 4, 5] print(len(nums)) # 5 print(sum(nums)) # 15 print(max(nums)) # 5 print(min(nums)) # 1

5.9 Рекурсія

Рекурсія — це коли функція викликає сама себе. Важливо: завжди має бути базова умова, щоб уникнути нескінченності.

def factorial(n): """Обчислення факторіала через рекурсію.""" if n == 0: return 1 return n * factorial(n-1) print(factorial(5)) # 120

5.10 Документація функцій (docstring)

Docstring — це текстова документація всередині функції. Використовується для пояснень роботи та параметрів. Переглядається через help().

def greet(name): """ Функція вітає користувача за іменем. Параметри: name (str): Ім'я користувача """ print(f"Привіт, {name}!") help(greet) # покаже документацію

ТЕМА 6: Модулі та імпорт в Python

Модулі дозволяють організовувати код у файли, робити його повторно використовуваним та легше підтримуваним.

6.1 Що таке модуль

Модуль — це будь-який Python-файл (*.py*), який містить змінні, функції або класи. Його призначення — розділяти код на логічні частини та повторно використовувати функції у різних програмах. Завдяки модулям код стає охайнішим, легшим для тестування та зрозумілішим.

# файл math_utils.py def add(a, b): return a + b def subtract(a, b): return a - b

6.2 Імпорт модуля

Щоб скористатися функціями з іншого файлу, його потрібно імпортувати. Імпорт завантажує модуль у програму і дає доступ до всіх його об’єктів.

import math_utils print(math_utils.add(5, 3)) # 8 print(math_utils.subtract(5, 3)) # 2

6.3 Імпорт окремих функцій

Можна імпортувати не весь модуль, а лише необхідні функції. Це робить код коротшим і зручнішим.

from math_utils import add print(add(10, 5)) # 15

6.4 Імпорт з псевдонімом (alias)

Псевдонім дозволяє скоротити ім’я модуля, що зручно у великих проєктах. Зазвичай використовують короткі позначення, наприклад np для numpy.

import math_utils as mu print(mu.add(2, 3)) # 5

6.5 Вбудовані модулі Python

Python містить сотні стандартних модулів, доступних без встановлення: math, random, os, datetime, json та інші. Вони допомагають працювати з математикою, файлами, системою, датами та випадковістю.

import math import random print(math.sqrt(16)) # 4.0 print(random.randint(1,5)) # випадкове число від 1 до 5

6.6 Модуль datetime

Модуль datetime дозволяє працювати з датами та часом: поточний час, форматування дати, різниця між датами тощо.

from datetime import datetime now = datetime.now() print(now) # поточна дата та час print(now.year) # рік print(now.month) # місяць print(now.day) # день

6.7 Модуль os

Модуль os дозволяє взаємодіяти з операційною системою: керувати файлами, папками, шляхами, отримувати поточну директорію.

import os print(os.getcwd()) # поточна директорія os.mkdir("test_folder") # створити папку

6.8 Модуль random

Модуль random використовується для генерації випадкових чисел, вибору випадкового елемента та перемішування списків.

import random print(random.randint(1, 10)) # випадкове число від 1 до 10 print(random.choice(["a","b","c"])) # випадковий елемент nums = [1,2,3] random.shuffle(nums) # перемішування списку print(nums)

6.9 Створення власного пакета

Пакет — це папка, що містить кілька модулів та файл __init__.py. Він дозволяє структурувати великий проєкт та імпортувати модулі з підпакетів.

mypackage/ __init__.py math_utils.py string_utils.py # Виклик функцій з пакета from mypackage.math_utils import add print(add(2,3)) # 5

6.10 Імпорт всіх функцій з модуля

Імпорт через * завантажує всі функції модуля. Але це не рекомендується — може створити конфлікт назв та ускладнити розуміння коду.

from math_utils import * print(add(4,5)) # 9 print(subtract(10,3)) # 7

ТЕМА 7: Файли та введення/виведення (I/O) в Python

Python дозволяє читати і записувати дані у файли, працювати з текстовими та бінарними файлами, а також отримувати ввід від користувача через клавіатуру.

7.1 Ввід від користувача

Функція input() зупиняє програму та чекає введення користувача з клавіатури. Введене завжди повертається як рядок (str).

name = input("Введіть своє ім'я: ") print("Привіт,", name)

7.2 Вивід на екран

Функція print() виводить текст, числа або змінні на екран. Можна форматувати рядки через f-рядки або конкатенацію.

age = 20 print("Вік:", age) print(f"Вік користувача: {age}")

7.3 Відкриття файлу

Файл відкривається через open(filename, mode), де mode визначає режим: "r" – читання, "w" – запис (перезапис), "a" – додавання.

file = open("example.txt", "w") file.write("Привіт, файл!\n") file.close()

7.4 Читання файлу

Можна прочитати весь файл одразу через read(), що повертає рядок із всім вмістом.

file = open("example.txt", "r") content = file.read() print(content) file.close()

7.5 Читання файлу рядок за рядком

Зручно для великих файлів. Цикл перебирає рядки, можна застосувати strip() для видалення пробілів і символів нового рядка.

with open("example.txt", "r") as file: for line in file: print(line.strip())

7.6 Запис у файл

Метод writelines() записує список рядків у файл. При цьому кожен рядок повинен закінчуватися символом нового рядка \n.

lines = ["Перший рядок\n", "Другий рядок\n"] with open("example.txt", "w") as file: file.writelines(lines)

7.7 Додаткові параметри відкриття файлу

Режим "a" додає рядки в кінець файлу, не перезаписуючи існуючий вміст. Також існують режим "rb" та "wb" для бінарного читання/запису.

with open("example.txt", "a") as file: file.write("Ще один рядок\n")

7.8 Робота з бінарними файлами

Бінарні файли містять дані у вигляді байтів. Використовуйте "rb" і "wb". Так можна працювати з зображеннями, аудіо та іншими непоточними даними.

with open("example.bin", "wb") as file: file.write(b'\x00\x01\x02\x03') with open("example.bin", "rb") as file: data = file.read() print(data)

7.9 Перевірка наявності файлу

Модуль os дозволяє перевіряти, чи існує файл або папка, перед тим як виконувати операції.

import os if os.path.exists("example.txt"): print("Файл існує") else: print("Файл не знайдено")

7.10 Видалення та перейменування файлів

Модуль os дозволяє: - os.rename() – перейменування файлу - os.remove() – видалення файлу

import os os.rename("example.txt", "new_example.txt") os.remove("new_example.txt")

ТЕМА 8: Об’єктно-орієнтоване програмування (ООП) в Python

ООП дозволяє створювати класи, об’єкти і використовувати такі концепції, як наслідування, поліморфізм, інкапсуляція.

8.1 Клас та об’єкт

Клас – це шаблон для створення об’єктів. Об’єкт – конкретний екземпляр класу. Об’єкти можуть мати атрибути (дані) та методи (функції).

class Person: def __init__(self, name, age): self.name = name # атрибут об’єкта self.age = age # створення об’єкта person1 = Person("Alice", 25) print(person1.name) # Alice print(person1.age) # 25

8.2 Метод __init__

Метод __init__ викликається автоматично під час створення об’єкта. Він використовується для ініціалізації атрибутів.

class Car: def __init__(self, brand, model): self.brand = brand self.model = model car1 = Car("Toyota", "Corolla") print(car1.brand) # Toyota print(car1.model) # Corolla

8.3 Атрибути об’єкта

Атрибути зберігають стан об’єкта. Можна додавати атрибути після створення об’єкта.

class Dog: def __init__(self, name, breed): self.name = name self.breed = breed dog1 = Dog("Buddy", "Labrador") print(dog1.name) # Buddy print(dog1.breed) # Labrador # Додавання нового атрибута dog1.age = 3 print(dog1.age) # 3

8.4 Методи об’єкта

Методи – це функції, що належать об’єкту. Вони можуть використовувати або змінювати атрибути.

class Dog: def __init__(self, name): self.name = name def bark(self): print(f"{self.name} гавкає!") dog1 = Dog("Buddy") dog1.bark() # Buddy гавкає!

8.5 Наслідування

Клас-нащадок успадковує атрибути та методи батьківського класу. Можна додавати нові методи або перевизначати існуючі.

class Animal: def eat(self): print("Тварина їсть") class Cat(Animal): # наслідування def meow(self): print("Кіт нявкає") cat1 = Cat() cat1.eat() # успадкований метод cat1.meow() # власний метод

8.6 Поліморфізм

Поліморфізм – здатність об’єктів різних класів відповідати на один і той же метод. Це дозволяє працювати з різними типами об’єктів однаково.

class Dog: def sound(self): print("Гав-гав") class Cat: def sound(self): print("Няв") animals = [Dog(), Cat()] for animal in animals: # виклик одного методу для різних об’єктів animal.sound()

8.7 Інкапсуляція

Інкапсуляція приховує дані об’єкта, роблячи їх недоступними ззовні. Приватні атрибути починаються з __. Доступ до них через методи класу.

class Person: def __init__(self, name, age): self.__name = name # приватний атрибут self.age = age def get_name(self): return self.__name # доступ через метод person1 = Person("Alice", 25) print(person1.get_name()) # Alice # print(person1.__name) -> помилка!

8.8 Метод __str__

Метод __str__ дозволяє виводити об’єкт у вигляді зрозумілого рядка, замість стандартного <object object at ...>.

class Person: def __init__(self, name, age): self.name = name self.age = age def __str__(self): return f"{self.name}, {self.age} років" person1 = Person("Alice", 25) print(person1) # Alice, 25 років

8.9 Статичні та класові методи

@staticmethod – метод без доступу до об’єкта чи класу, викликається через клас. @classmethod – метод, що отримує клас як перший аргумент cls.

class Math: name = "Math" @staticmethod def add(a, b): return a + b @classmethod def info(cls): print(f"Це клас {cls.name}") print(Math.add(5, 3)) # 8 Math.info() # Це клас Math

8.10 Видалення об’єктів та атрибутів

За допомогою del можна видаляти атрибути об’єкта або сам об’єкт. Якщо об’єкт видаляється, всі посилання на нього втрачаються.

class Person: def __init__(self, name): self.name = name person1 = Person("Alice") del person1.name # видалення атрибута # del person1 # видалення об’єкта

ТЕМА 9: Модулі, пакети та бібліотеки в Python

Модулі та пакети дозволяють організовувати код, робити його повторно використовуваним та структурованим, а бібліотеки надають готові функції та класи для різних задач.

9.1 Модуль у Python

Модуль — це файл з розширенням .py, що містить Python-код (функції, змінні, класи), який можна повторно використовувати в інших програмах. Використання модулів дозволяє структурувати код і уникати дублювання.

# файл my_module.py def greet(name): return f"Привіт, {name}!" def add(a, b): return a + b PI = 3.14159
# головний файл import my_module print(my_module.greet("Alice")) # Привіт, Alice! print(my_module.add(5, 3)) # 8 print(my_module.PI) # 3.14159

9.2 Імпорт модулів

Для використання модуля в іншому файлі застосовується ключове слово import. Можна імпортувати весь модуль або окремі елементи.

from my_module import greet, add print(greet("Bob")) # Привіт, Bob! print(add(10, 7)) # 17

9.3 Стандартна бібліотека Python

Python постачає велику стандартну бібліотеку, що включає модулі для роботи з файлами, датами, математикою, випадковими числами, системними операціями та інше.

import os import math import random print(os.getcwd()) # поточна директорія print(math.sqrt(25)) # 5.0 print(random.randint(1, 10)) # випадкове число від 1 до 10

9.4 Пакети

Пакет — це каталог, що містить модулі та файл __init__.py. Пакети дозволяють організувати модулі у ієрархічну структуру.

# структура my_package/ │ ├─ __init__.py ├─ module1.py └─ module2.py # головний файл from my_package.module1 import func1 func1()

9.5 Встановлення зовнішніх бібліотек

Бібліотеки, яких немає у стандартній бібліотеці, можна встановлювати через pip.

# приклад з бібліотекою requests import requests response = requests.get("https://api.github.com") print(response.status_code) print(response.json())

9.6 Використання бібліотеки NumPy

NumPy — бібліотека для наукових обчислень. Основна структура даних — масив ndarray.

import numpy as np arr = np.array([1, 2, 3, 4]) print(arr * 2) # [2 4 6 8] print(arr + 5) # [6 7 8 9] print(np.mean(arr)) # 2.5

9.7 Використання бібліотеки Pandas

Pandas використовується для роботи з таблицями і серіями даних.

import pandas as pd data = {'Name': ['Alice', 'Bob'], 'Age': [25, 30]} df = pd.DataFrame(data) print(df) print(df['Name']) # стовпець 'Name' print(df.iloc[0]) # перший рядок

9.8 Використання бібліотеки Matplotlib

Matplotlib — бібліотека для візуалізації даних.

import matplotlib.pyplot as plt x = [1, 2, 3, 4] y = [10, 20, 25, 30] plt.plot(x, y, marker='o') plt.title("Графік") plt.xlabel("X") plt.ylabel("Y") plt.grid(True) plt.show()

9.9 Використання бібліотеки Requests

Requests спрощує роботу з HTTP-запитами.

import requests response = requests.get("https://api.github.com") if response.status_code == 200: data = response.json() print(data) else: print("Помилка запиту")

9.10 Створення власного пакета

Власний пакет дозволяє організувати модулі у проекті. Структура: каталог з __init__.py та модулями. Імпорт з пакета можна здійснювати через точкову нотацію.

# структура my_package/ ├─ __init__.py ├─ greetings.py # greetings.py def hello(name): return f"Hello, {name}!" # головний файл from my_package.greetings import hello print(hello("Alice")) # Hello, Alice!

ТЕМА 10: Робота з файлами та базами даних у Python

Python дозволяє читати та записувати файли, працювати з текстовими, CSV, JSON файлами, а також підключатися до баз даних.

10.1 Робота з текстовими файлами — відкриття та закриття

Текстові файли відкриваються за допомогою open(filename, mode). Режими: 'r' — читання, 'w' — запис, 'a' — додавання, 'x' — створення нового файлу. Після роботи обов’язково закривати файл через file.close().

# запис у файл file = open("example.txt", "w") file.write("Привіт, Python!\n") file.close() # читання з файлу file = open("example.txt", "r") content = file.read() print(content) file.close()

10.2 Робота з файлами за допомогою with

Блок with забезпечує автоматичне закриття файлу.

with open("example.txt", "r") as file: content = file.read() print(content)

10.3 Читання файлів построчно

with open("example.txt", "r") as file: for line in file: print(line.strip())

10.4 Запис у файл построчно

lines = ["Перша лінія\n", "Друга лінія\n", "Третя лінія\n"] with open("example.txt", "a") as file: file.writelines(lines)

10.5 Робота з CSV файлами

import csv # запис у CSV with open("data.csv", "w", newline="") as file: writer = csv.writer(file) writer.writerow(["Name", "Age"]) writer.writerow(["Alice", 25]) writer.writerow(["Bob", 30]) # читання CSV with open("data.csv", "r") as file: reader = csv.reader(file) for row in reader: print(row)

10.6 Робота з JSON файлами

import json data = {"name": "Alice", "age": 25} # запис у JSON with open("data.json", "w") as file: json.dump(data, file) # читання з JSON with open("data.json", "r") as file: loaded_data = json.load(file) print(loaded_data["name"])

10.8 Підключення до SQLite бази даних

import sqlite3 conn = sqlite3.connect("my_database.db") cursor = conn.cursor() cursor.execute(""" CREATE TABLE IF NOT EXISTS users ( id INTEGER PRIMARY KEY, name TEXT, age INTEGER ) """) conn.commit() conn.close()

10.9 Додавання та читання даних у базі

conn = sqlite3.connect("my_database.db") cursor = conn.cursor() # додавання даних cursor.execute("INSERT INTO users (name, age) VALUES (?, ?)", ("Alice", 25)) cursor.execute("INSERT INTO users (name, age) VALUES (?, ?)", ("Bob", 30)) conn.commit() # читання даних cursor.execute("SELECT * FROM users") rows = cursor.fetchall() for row in rows: print(row) conn.close()
