ТЕМА 1: Основи CSS

1.1 Вступ до CSS

CSS (Cascading Style Sheets) — це мова стилів для опису вигляду веб-сторінок. Використовується для:

  • оформлення кольорів, шрифтів та фонів
  • визначення розмірів елементів та відступів
  • створення адаптивного дизайну
  • анімованих ефектів та переходів

Приклад простого CSS-коду:

p { color: blue; font-size: 16px; }

1.2 Синтаксис CSS

CSS складається з правил (rules), що містять селектори та властивості:

  • Селектор — елемент HTML, до якого застосовується стиль
  • Властивості — характеристики стилю (color, font-size)
  • Значення — конкретне значення властивості
h1 { color: red; text-align: center; }

1.3 Коментарі CSS

Коментарі використовуються для пояснення коду та не виконуються браузером:

/* Це коментар */

1.4 Селектори

Селектори визначають, які елементи HTML будуть стилізовані:

  • Типові селектори: p, h1, div
  • Класи: .class-name
  • ID: #id-name
  • Універсальний селектор: *
.highlight { background-color: yellow; }

1.5 Атрибути та псевдокласи

CSS дозволяє стилізувати елементи за атрибутами та станами:

  • Псевдокласи: a:hover — при наведенні
  • Атрибутні селектори: input[type="text"]
a:hover { color: red; }

1.6 Псевдоелементи

Псевдоелементи дозволяють стилізувати частини елементів:

  • ::before — вставляє вміст перед елементом
  • ::after — вставляє вміст після елементу
p::before { content: ">> "; }

1.7 Спадкування властивостей

Деякі властивості передаються від батьків до дочірніх елементів:

body { font-family: Arial; } p { color: black; /* наслідує font-family з body */ }

1.8 Специфічність селекторів

Якщо один елемент стилізований кількома правилами, застосовується правило з вищою специфічністю:

#id-name { color: red; } .class-name { color: blue; } p { color: black; } /* Пріоритет: ID > клас > тег */

1.9 !important

Додає максимальний пріоритет стилю:

p { color: green !important; }

1.10 Приклад CSS

Об'єднання всіх знань для стилізації сторінки:

body { font-family: 'Arial', sans-serif; background-color: #f0f0f0; } h1 { color: #333; text-align: center; } p { color: #555; line-height: 1.5; }

ТЕМА 2: Основні властивості та структури CSS

2.1 Кольори CSS

CSS дозволяє задавати кольори для тексту, фону та меж елементів. Можна використовувати:

  • Назви кольорів: red, blue, green
  • HEX-коди: #ff0000, #00ff00
  • RGB/RGBA: rgb(255,0,0), rgba(0,0,255,0.5)
  • HSL: hsl(120, 100%, 50%)
p { color: #ff0000; background-color: rgb(200,200,200); }

2.2 Шрифти CSS

Можна задавати сімейство шрифтів, розмір, стиль та товщину:

h1 { font-family: 'Arial', sans-serif; font-size: 36px; font-weight: bold; font-style: italic; }

2.3 Відступи та padding/margin

CSS дозволяє відступи всередині елементу (padding) та зовні (margin):

div { margin: 20px; padding: 10px; border: 1px solid #000; }

2.4 Висота та ширина

Встановлюємо розміри блоків:

div { width: 200px; height: 100px; }

2.5 Модель Box

Кожен елемент має модель box: content → padding → border → margin.

div { width: 100px; padding: 10px; border: 5px solid black; margin: 15px; }

2.6 Властивості display

Визначає, як елемент відображається:

  • block — блочний
  • inline — в рядок
  • inline-block — inline + можливість задати розмір
  • none — приховує елемент
span { display: inline-block; width: 100px; }

2.7 Позиціювання

CSS дозволяє керувати позицією елементів:

  • static — за замовчуванням
  • relative — відносно свого звичайного місця
  • absolute — відносно батьківського блоку з позицією
  • fixed — фіксовано на екрані
div { position: absolute; top: 50px; left: 100px; }

2.8 Overflow

Керування вмістом, який виходить за межі блоку:

div { width: 150px; height: 100px; overflow: auto; }

2.9 Float та Clear

Позиціювання елементів ліворуч або праворуч:

img { float: left; margin-right: 10px; } .clearfix::after { content: ""; display: block; clear: both; }

2.10 Непрозорість

Контролює прозорість елемента:

div { opacity: 0.5; }

ТЕМА 3: Умови та розгалуження в CSS

3.1 @media — основа умов

У CSS умовні правила застосовуються за допомогою медіа-запитів. Вони дозволяють змінювати стиль залежно від розміру екрана, типу пристрою тощо.

@media (max-width: 768px) { body { background-color: lightblue; } }

3.2 @media для конкретної умови

Можна застосовувати стилі лише за певних умов:

@media (min-width: 1024px) { .container { width: 80%; } }

3.3 Багато умов у медіа-запитах

Об’єднання умов через and:

@media (min-width: 768px) and (max-width: 1024px) { .menu { display: flex; } }

3.4 Логічне "або" у медіа

Застосування стилю, якщо виконана хоча б одна умова:

@media (max-width: 480px), (orientation: portrait) { .sidebar { display: none; } }

3.5 Вкладені умови

Можна використовувати вкладені селектори всередині медіа-запитів:

@media (max-width: 768px) { .container { .menu { display: none; } } }

3.6 Порівняння властивостей

CSS не має операторів порівняння, але можна використовувати різні значення для медіа-запитів та псевдокласів:

button:hover { background-color: green; }

3.7 Використання числових умов

Медіа-запити дозволяють перевіряти ширину екрана та інші параметри:

@media (min-width: 600px) { .box { width: 400px; } }

3.8 Коротка форма через коми

Декілька умов можна об’єднувати через кому:

@media (max-width: 500px), (orientation: portrait) { .header { display: none; } }

3.9 Використання prefers-color-scheme

CSS дозволяє змінювати теми залежно від налаштувань пристрою:

@media (prefers-color-scheme: dark) { body { background-color: #222; color: #fff; } }

3.10 Умови в одній лінії (псевдокласи)

Псевдокласи дозволяють змінювати стиль елементу за умовами взаємодії:

a:hover { color: red; } input:focus { border-color: blue; }

ТЕМА 4: Цикли в CSS

4.1 @for — перебір елементів колекцій

CSS сам по собі не підтримує цикли, але препроцесори (SASS, LESS) дозволяють використовувати @for для генерації стилів:

@for $i from 1 through 5 { .col-#{$i} { width: 20% * $i; } }

4.2 @each — перебір списку

Препроцесори дозволяють перебирати списки елементів:

@each $color in red, green, blue { .text-#{$color} { color: $color; } }

4.3 Перебір рядків

Можна перебирати набори класів для стилізації схожих елементів:

@for $i from 1 through 3 { .item-#{$i}::before { content: "Item #{$i}"; } }

4.4 Перебір словника

У SASS можна перебирати мапи:

$themes: (primary: blue, secondary: green); @each $name, $color in $themes { .btn-#{$name} { background-color: $color; } }

4.5 @while — виконується поки умова True

Препроцесори дозволяють створювати цикли з умовами:

$i: 1; @while $i <= 3 { .item-#{$i} { width: 100px * $i; } $i: $i + 1; }

4.6 Нескінчений цикл (while true)

У CSS/SASS небезпечні нескінченні цикли не рекомендуються, але синтаксис:

$i: 1; @while true { .item-#{$i} { width: 50px * $i; } $i: $i + 1; @if $i > 10 { @break; } }

4.7 break — зупиняє цикл

Для виходу з циклу використовується @break:

@for $i from 1 through 10 { @if $i == 5 { @break; } .item-#{$i} { width: 20px * $i; } }

4.8 continue — пропускає ітерацію

Пропуск ітерації виконується через @continue:

@for $i from 1 through 5 { @if $i == 3 { @continue; } .item-#{$i} { width: 20px * $i; } }

4.9 else після циклу

У CSS/SASS немає прямого аналогу else після циклу, але можна використовувати умовні правила всередині циклу.

4.10 Вкладені цикли

Використання циклів всередині циклів для генерації складних структур:

@for $i from 1 through 3 { @for $j from 1 through 2 { .item-#{$i}-#{$j} { width: 20px * $j; } } }
