# CodeCraft-Site1
Сайт для вивчення мов програмування — це вебплатформа, створена відповідно до принципів програмної інженерії, що забезпечує систематичне, надійне та якісне навчання користувачів. Платформа надає інтерактивні уроки, практичні завдання, тести, систему прогресу та інструменти для самостійного навчання.
Продукт розроблений як програмна система, що складається з фронтенду, бекенду, бази даних та сервісів інтерактивного виконання коду.
Вимоги до системи:
Функціональні вимоги
- Реєстрація та авторизація користувачів
- Перегляд курсів (Python, C++, JavaScript тощо)
- Проходження уроків з теорією та прикладами
- Інтерактивний редактор коду
- Тестові завдання та автоматична перевірка
- Панель прогресу
- Система рекомендацій наступних уроків
- Адмін-панель для додавання нових курсів
Нефункціональні вимоги
- Продуктивність: сторінки завантажуються до 2 секунд
- Надійність: система працює без відмов, MTTF високий
- Безпека: аутентифікація, шифрування паролів, захист від SQL-ін’єкцій
- Масштабованість: можливість додавати нові курси без зміни архітектури
- Юзабіліті: інтуїтивний інтерфейс, адаптивний дизайн
