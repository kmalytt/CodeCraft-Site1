CodeCraft — Платформа для вивчення мов програмування
CodeCraft — це вебплатформа для інтерактивного вивчення мов програмування, створена відповідно до принципів програмної інженерії та стандартів життєвого циклу програмного забезпечення. Проєкт розроблений у рамках дисципліни “Software Engineering Fundamentals” і демонструє повний цикл створення сучасної навчальної системи: від визначення вимог до супроводу та розвитку продукту.

Призначення та мета системи
Метою CodeCraft є забезпечення доступного, структурованого та практичного навчання програмуванню. Платформа дозволяє користувачам вивчати популярні мови (Python, C++, JavaScript), проходити інтерактивні уроки, виконувати завдання, запускати код у браузері та відстежувати власний прогрес.
Система орієнтована на:
- студентів і початківців у програмуванні
- викладачів, які хочуть використовувати платформу як навчальний інструмент
- користувачів, які прагнуть самостійно опанувати програмування

Функціональні можливості
Для користувачів
- реєстрація та авторизація
- вибір курсу та проходження уроків
- інтерактивний редактор коду з можливістю запуску
- тестові завдання з автоматичною перевіркою
- система прогресу та рекомендацій
- адаптивний інтерфейс
Для адміністратора
- додавання та редагування курсів
- створення уроків, тестів і практичних завдань
- керування користувачами

Нефункціональні характеристики
Система відповідає вимогам ISO/IEC 25010 та включає такі атрибути якості:
- Надійність: стабільна робота, обробка помилок, логування
- Продуктивність: швидке завантаження сторінок, оптимізовані запити
- Безпека: хешування паролів, захист від SQL-ін’єкцій, контроль доступу
- Юзабіліті: інтуїтивний інтерфейс, адаптивний дизайн
- Масштабованість: можливість додавання нових курсів без зміни архітектури
- Супровідність: чистий код, модульна структура, документація

Архітектура системи
Платформа побудована за клієнт-серверною архітектурою.
Frontend
- HTML, CSS, JavaScript
- Статичні сторінки + інтерактивні компоненти
- Адаптивний дизайн
Backend
- Python (Flask або FastAPI)
- REST API для взаємодії з фронтендом
- Обробка логіки уроків, тестів, авторизації
Database
- SQLite / PostgreSQL
- Таблиці: Users, Courses, Lessons, Tests, Progress
Code Execution Service
- окремий модуль для безпечного запуску коду користувача

Модель життєвого циклу
Для розробки CodeCraft застосовано інкрементну модель, що дозволила створювати систему поетапно:
- Ітерація 1: базова структура, авторизація, головна сторінка
- Ітерація 2: уроки та перегляд контенту
- Ітерація 3: інтерактивний редактор коду
- Ітерація 4: тести та система прогресу
- Ітерація 5: адмін-панель, оптимізація, рефакторинг
Кожна ітерація включала аналіз вимог, проектування, реалізацію, тестування та реліз.

Тестування
У проєкті застосовано різні рівні та види тестування:
- Модульне тестування: перевірка API
- Інтеграційне: взаємодія фронтенду з бекендом
- Системне: повна перевірка функціональності
- Регресійне: після кожного оновлення
- Юзабіліті-тестування: оцінка зручності інтерфейсу
Метрики тестування включають:
- покриття тестами
- кількість дефектів на реліз
- час відповіді API

Забезпечення якості
Система розроблена з урахуванням:
- стандартів ISO/IEC 25010
- принципів чистого коду
- контролю версій (Git)
- документування API та структури проєкту

Супровід та розвиток
Після впровадження система підтримує всі види обслуговування:
- Коригувальне: виправлення помилок
- Адаптивне: оновлення під нові браузери та технології
- Попереджувальне: рефакторинг, оптимізація
- Розширювальне: додавання нових курсів, модулів, функцій
Платформа підтримує реінжиніринг, що дозволяє модернізувати архітектуру та переносити систему на нові технології.

Висновок
CodeCraft — це сучасна навчальна платформа, створена відповідно до принципів програмної інженерії та стандартів життєвого циклу ПЗ. Вона поєднує інтерактивне навчання, практичні інструменти та гнучку архітектуру, що робить її ефективним рішенням для студентів, викладачів і всіх, хто хоче опанувати програмування.
